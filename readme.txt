== Description ==

This plugin provides four different Pop up Like Boxes which appears on hover on little social icons that are at left sidebar on your blog or website.

One of the best ways to increase your Social fans is to add the jQuery Pop-up Social Box to your Blog. Once this widget is installed on your blog, your visitors will be shown a slide out box with the option to like your Facebook Page, or want to follow on Twitter, or want to add you in there Google+ circle or subscribe with your YouTube channel.

Special features:

- 4 in 1 social widgets. Facebook, Twitter, Google Plus & Youtube.
- Eye caching flat style.
- All major device supported.
- Interactive jQuery slide out animation.
- Super easy setup through admin setting panel.
- 100% working in small devices.
- Very lightweight only 17KB.

== Installation ==

1. Upload `pure-theme-slide-social-tabs` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin.
3. Use "PT Social Tabs" from wordpress admin menu to configure social profiles. (Please Press "Save Changes")

Admin panel setup guide:

- Facebook Page

Login to facebook account, go to <a href="https://www.facebook.com/username" terget="_blank">facebook.com/username</a>.Then select a page. Now a link will be appeared like www.facebook.com/USERNAME . Copy USERNAME only & paste it in admin setting sections.

- Twitter Profile

Login to twitter account, go to <a href="https://twitter.com/settings/account" terget="_blank">twitter.com/settings/account</a>. In the account section twitter username box will be shown like twitter.com/USERNAME. Copy USERNAME only & paste it in admin setting sections.

Important Note: After doing that you will find a configuration Notice in twitter widget like " Generate Code @USERNAME ". Click on it. A new website will come to your browser. Just Signin with your twitter account, ignore the other settings there. Job done !!!

- Google Plus ID

Login to google plus account, go to <a href="https://plus.google.com/" terget="_blank">plus.google.com</a>. Now go to profile . In the browser url section google plus ID will be found like https://plus.google.com/109780552166432187391 . Copy ID only & paste it in admin setting sections.

- Youtube Channel

Login to youtube account, go to <a href="https://www.youtube.com/user/" terget="_blank">www.youtube.com/user</a>. In the browser url section youtube channel name will be found like www.youtube.com/user/CHANNELNAME . Copy ID only & paste it in admin setting sections.
